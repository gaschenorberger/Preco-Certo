DOCUMENTAÇÃO DE INSTALAÇÃO FRONTEND

1. Instalar node.js (https://nodejs.org/en/download)
2. Após isso, digite 'npm install' no terminal.

2.1 Caso dê erro ('npm : O arquivo C:\Program Files\nodejs\npm.ps1 não pode ser carregado porque a execução de scripts foi desabilitada neste sistema.')
2.2 Abra o powerShell como ADMIN e digite ('Set-ExecutionPolicy RemoteSigned -Scope CurrentUser') - (Deseja alterar a política de execução?) - (S)
2.3 Tente 'npm install' novamente.

3. Execute no terminal 'cd frontend'
4. Após isso, execute 'npm start'

5. PRONTO, após todos esses passos o sistema FRONTEND já vai ter iniciado no seu computador.
